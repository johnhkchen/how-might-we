## BAML + Go Backend Foundation

Stand up the Go backend with BAML integration and get the first endpoint (RefinePersona) streaming structured output via SSE. This validates the core technical stack: BAML generating Go code, structured streaming over SSE, and the handler pattern that all subsequent endpoints will follow.

### Track Breakdown

**Single track** — these are sequential since each builds on the previous.

1. **T-001-01**: Initialize Go module, install BAML dependency, generate Go client from the scaffolded .baml files
2. **T-001-02**: Implement the SSE streaming handler pattern and wire up the `/api/persona` endpoint
3. **T-001-03**: Verify persona endpoint works end-to-end with `curl --no-buffer`

### DAG

```
T-001-01 → T-001-02 → T-001-03
```
