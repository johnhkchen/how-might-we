## Context

Both the Lambda Function URL CORS config and the CF Worker proxy currently allow `*` origins. In production, these should be restricted to the CF Pages domain (and eventually the custom domain). The `FRONTEND_URL` is already stored in Doppler.

## Acceptance Criteria

- [ ] CF Worker `ALLOWED_ORIGIN` set to the CF Pages production URL (from Doppler `FRONTEND_URL`)
- [ ] Lambda Function URL CORS `AllowOrigins` restricted to the same origin
- [ ] CORS preflight still works from the allowed origin
- [ ] Requests from other origins are rejected with appropriate CORS error
- [ ] Local development still works (Go middleware handles CORS for localhost)
- [ ] Document how to add additional origins (e.g., custom domain) when needed
