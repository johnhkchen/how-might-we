## Context

Wire up the workshop page (`/workshop`) to orchestrate the full stage flow: Setup → Analyze → Expand → Refine (loop) → Export. Stages appear progressively as the user advances. The refine loop (Stage 4) repeats when the user clicks "Go Deeper" — each iteration passes the full session state to `/api/refine`.

## Acceptance Criteria

- [ ] Workshop page shows Stage 1 (Setup) initially
- [ ] Completing Stage 1 reveals Stage 2 (Analyze)
- [ ] Completing Stage 2 reveals Stage 3 (Expand) with "Expand" button
- [ ] After expansion, variant cards appear with select/skip/edit/clip actions
- [ ] "Go Deeper" button triggers refine loop (Stage 4) — calls `/api/refine` with full session
- [ ] New variants from refine appear visually distinct from previous ones
- [ ] Iteration count displayed and incremented each cycle
- [ ] Previous stages remain visible and editable (editing persona shows warning about re-analysis)
- [ ] All stages work with mock API (`npm run dev:mock`)
- [ ] Playwright tests cover the full flow with mock data
