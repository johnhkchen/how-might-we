## Context

Implement the `/api/analyze` endpoint using the handler pattern from T-001-02. This endpoint takes a raw HMW statement plus a `ProblemContext` and streams back an `HMWAnalysis`.

## Acceptance Criteria

- [ ] `handleAnalyze` parses `{ "statement": string, "context": ProblemContext }` from request body
- [ ] Calls BAML `Stream.AnalyzeHMW()` and streams partial `HMWAnalysis` objects
- [ ] Returns meaningful analysis: implicit user, embedded assumptions, scope level, underlying tension
- [ ] Testable via `curl` with output from persona endpoint as context input
