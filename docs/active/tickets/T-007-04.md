## Context

`moveLabel()` (converts `"shifted_user"` → `"Shifted User"`) and `moveColors` (maps move types to Tailwind color classes) are duplicated across VariantCard, VariantGrid, ClipBoard, and ExportPanel. Extract to a shared utility.

## Acceptance Criteria

- [ ] `moveLabel()` and `moveColors` extracted to `$lib/utils/moves.ts`
- [ ] All components import from the shared utility instead of defining locally
- [ ] No behavior change — same labels, same colors
- [ ] All Playwright tests pass
- [ ] `npm run check` and `npm run lint` pass
