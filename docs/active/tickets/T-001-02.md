## Context

Implement the SSE streaming handler pattern in `handlers.go` and wire up the `/api/persona` endpoint as the first real endpoint. This establishes the pattern all other endpoints will follow: parse JSON input, set SSE headers, call BAML streaming function, flush partials, send `[DONE]` sentinel.

## Acceptance Criteria

- [ ] `handlePersona` parses `{ "rawInput": string }` from request body
- [ ] Sets correct SSE headers: `Content-Type: text/event-stream`, `Cache-Control: no-cache`, `Connection: keep-alive`
- [ ] Calls BAML `Stream.RefinePersona()` with `context.Context` from request
- [ ] Streams each partial as `data: {json}\n\n` and flushes after each
- [ ] Sends `data: [DONE]\n\n` when stream completes
- [ ] Returns 400 on invalid input, 500 on BAML errors
- [ ] Handler pattern is extractable for reuse by other endpoints
