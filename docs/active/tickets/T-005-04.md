## Context

Add visual polish to the streaming UX. When BAML is streaming, the UI should feel like watching someone think. This includes transitions for fields appearing, cards entering the grid, and subtle loading indicators.

## Acceptance Criteria

- [ ] PersonaCard fields fade/slide in as they arrive during streaming
- [ ] AnalysisPanel sections appear with subtle reveal animations
- [ ] VariantCards enter the grid with a staggered animation (each card slightly delayed)
- [ ] Skeleton/shimmer placeholders shown for fields that haven't arrived yet
- [ ] "Streaming..." indicator with subtle pulse animation while API call is active
- [ ] Animations are smooth (use CSS transitions/animations, avoid layout thrash)
- [ ] Animations respect `prefers-reduced-motion` media query
