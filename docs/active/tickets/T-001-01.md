## Context

The Go backend has scaffolded `.baml` files and a `go.mod`, but the BAML Go client hasn't been generated yet. This ticket gets the Go module properly initialized with all dependencies resolved and the BAML client generated and compiling.

## Acceptance Criteria

- [ ] `go mod tidy` resolves all dependencies including `github.com/boundaryml/baml`
- [ ] `baml-cli generate` runs successfully and produces `backend/baml_client/` directory
- [ ] `go build ./...` compiles without errors
- [ ] Generated client exposes typed functions: `RefinePersona`, `AnalyzeHMW`, `ExpandHMW`, `RefineHMW`
- [ ] Generated client exposes streaming variants via `Stream` package
