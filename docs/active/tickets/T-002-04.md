## Context

Verify the complete backend flow works end-to-end: persona → analyze → expand → refine. Chain the outputs together manually via curl to confirm the full pipeline. Optionally write a shell script or Go test that automates this.

## Acceptance Criteria

- [ ] Full flow tested: persona output feeds into analyze, analyze output feeds into expand, expand output (with candidate statuses set) feeds into refine
- [ ] Each endpoint returns valid, well-structured JSON matching the BAML types
- [ ] SSE streaming works correctly at each stage (partials build up, [DONE] terminates)
- [ ] Error handling works: invalid JSON returns 400, missing fields return meaningful errors
- [ ] CORS preflight (OPTIONS) returns correct headers
