## Context

Enhance the rate limiting in the CF Worker beyond the basic IP-based limit. Add per-session limits and consider using Cloudflare's built-in rate limiting rules for production.

## Acceptance Criteria

- [ ] Per-IP rate limit: 20 requests per minute
- [ ] Per-session rate limit: 50 API calls per session (tracked via session token)
- [ ] Rate limit headers in responses (`X-RateLimit-Remaining`, `X-RateLimit-Reset`)
- [ ] 429 response with retry-after when limit exceeded
- [ ] Limits configurable via worker environment variables
