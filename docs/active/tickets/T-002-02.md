## Context

Implement the `/api/expand` endpoint. Takes an `HMWAnalysis` and `ProblemContext`, streams back an `HMWExpansion` with 6-8 diverse variant reframings.

## Acceptance Criteria

- [ ] `handleExpand` parses `{ "analysis": HMWAnalysis, "context": ProblemContext }` from request body
- [ ] Calls BAML `Stream.ExpandHMW()` and streams partial `HMWExpansion` objects
- [ ] Variants stream in one by one (partial array builds up)
- [ ] Each variant has a distinct `move` type and clear rationale
- [ ] `emergentTheme` appears in final partial
