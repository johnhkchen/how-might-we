## Context

Verify the persona endpoint works end-to-end by running the Go server locally and testing with `curl --no-buffer`. This confirms BAML streaming works through the Go HTTP handler to the client. Requires a valid `ANTHROPIC_API_KEY` environment variable.

## Acceptance Criteria

- [ ] `go run main.go` starts server on `:8080` without errors
- [ ] `curl --no-buffer -X POST http://localhost:8080/api/persona -H 'Content-Type: application/json' -d '{"rawInput":"junior designer at a SaaS company"}'` returns streaming SSE events
- [ ] Each SSE event contains a progressively more complete `Persona` JSON object
- [ ] Final event before `[DONE]` contains a complete Persona with all required fields
- [ ] CORS headers are present in response
