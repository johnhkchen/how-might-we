## Context

Get the frontend dev environment fully working: install dependencies, verify the SvelteKit dev server starts, confirm Tailwind CSS is processing, ESLint and Prettier run, and Playwright is installed. The scaffolding is in place — this ticket makes it actually run.

## Acceptance Criteria

- [ ] `npm install` completes without errors
- [ ] `npm run dev` starts the dev server on `:5173`
- [ ] Landing page renders with Tailwind styles applied
- [ ] `npm run lint` runs ESLint without configuration errors
- [ ] `npm run format:check` runs Prettier without configuration errors
- [ ] `npx playwright install chromium` installs the browser
- [ ] `npm test` runs the basic Playwright tests and they pass
- [ ] `npm run dev:mock` starts with mock API mode enabled
