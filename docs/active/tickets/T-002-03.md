## Context

Implement the `/api/refine` endpoint. Takes a full `HMWSession` state (including candidates with their statuses) and streams back `HMWRefinement` with new variants, tensions, and recommendations.

## Acceptance Criteria

- [ ] `handleRefine` parses `{ "session": HMWSession }` from request body
- [ ] Calls BAML `Stream.RefineHMW()` and streams partial `HMWRefinement` objects
- [ ] New variants reflect the direction of selected/edited candidates
- [ ] Tensions and recommendations appear in later partials
- [ ] `suggestedNextMove` provides actionable guidance for the next iteration
