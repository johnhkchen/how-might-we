## Context

After all token efficiency changes (S-008) land, verify the full flow works end-to-end locally and deployed. Rebuild and redeploy all three components.

## Steps

1. **Local verification**
   - `cd backend && doppler run -- go run main.go`
   - Test all 4 endpoints with curl, verify outputs are concise
   - Verify persona uses Haiku (faster response, check logs)
   - Verify analysis returns 3 reframe options
   - Verify expansion returns 4 variants
   - `cd frontend && npm run dev:mock` — verify UI with updated fixtures
   - `npm run check && npm run lint` — no errors
   - `npx playwright test` — all tests pass

2. **Build and deploy**
   - `cd backend && ./build.sh`
   - Copy bundle to `/tmp/hmw-bundle`
   - `npx sst deploy --stage dev`
   - Verify RESPONSE_STREAM invoke mode
   - `cd frontend && doppler run -- bash -c 'PUBLIC_API_URL=$PUBLIC_API_URL npm run build'`
   - `npx wrangler pages deploy .svelte-kit/cloudflare --project-name hmw-workshop`

3. **Production verification**
   - Test full flow on https://hmw-workshop.pages.dev
   - Confirm streaming works for all stages
   - Confirm reframe options appear and are selectable
   - Confirm variant count is 4 per expansion

## Acceptance Criteria

- [ ] All Playwright tests pass locally
- [ ] Backend builds and deploys without errors
- [ ] Frontend builds and deploys without errors
- [ ] Full workshop flow works on live site: setup → analyze (3 options) → expand (4 variants) → refine → export
- [ ] No regressions in streaming UX
- [ ] Persona responses noticeably faster (Haiku)
- [ ] All outputs visibly more concise than before
