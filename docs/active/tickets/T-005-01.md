## Context

Build the ExportPanel component (Stage 5). Takes the clipped HMWs from the ClipBoard and formats them for export in three formats: plain text list, structured markdown (with persona, context, and rationale per HMW), and JSON.

## Acceptance Criteria

- [ ] ExportPanel shows clipped HMWs with export format selector
- [ ] **Plain text**: Numbered list of HMW statements
- [ ] **Markdown**: Includes persona summary, domain, each HMW with its move type and rationale
- [ ] **JSON**: Full structured export matching the `HMWSession` type (parseable by other tools)
- [ ] Copy-to-clipboard button for each format
- [ ] Download button generates a file (`.txt`, `.md`, or `.json`)
- [ ] Empty state when no HMWs are clipped
