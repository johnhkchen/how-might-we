## Context

Create a Cloudflare Worker that sits in front of the Lambda Function URL. Handles CORS (replacing the Go middleware in production), basic rate limiting, and Turnstile verification placeholder. The worker proxies requests to the Lambda URL and streams responses back.

## Acceptance Criteria

- [ ] Worker script created in project (e.g., `worker/` directory)
- [ ] Proxies POST requests to Lambda Function URL
- [ ] Passes through SSE streaming responses without buffering
- [ ] Handles CORS preflight and response headers
- [ ] Basic rate limiting by IP (e.g., 10 requests/minute)
- [ ] Deployable via `wrangler deploy` or integrated into SST config
