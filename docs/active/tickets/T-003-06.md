## Context

Build the Stage 3-4 (Expand + Refine) UI components: VariantCard, VariantGrid, and ClipBoard. VariantCards appear one at a time as they stream in, each showing the HMW statement, move type tag, and rationale. Users can select, skip, edit, or clip each card. ClipBoard is a persistent panel showing the "shopping cart" of clipped HMWs.

## Acceptance Criteria

- [ ] VariantCard shows: HMW statement, move type tag (color-coded), rationale text
- [ ] VariantCard actions: Select (highlight), Skip (gray out), Edit (inline), Clip (add to clipboard)
- [ ] Actions update candidate status in session store
- [ ] VariantGrid displays cards in a responsive grid/list layout
- [ ] New cards animate in during streaming
- [ ] Move type tags are color-coded: narrowed (green), broadened (purple), shifted_user (orange), reframed_constraint (teal), elevated_abstraction (indigo), inverted (red), combined (amber), decomposed (sky)
- [ ] ClipBoard panel shows all clipped HMWs with remove option
- [ ] ClipBoard is persistent (visible alongside the variant grid)
- [ ] ClipBoard shows count badge
